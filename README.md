## 🚀 What this app does

ui-style-extractor helps you turn a UI page into a simple style guide. It reads a page, pulls out the design parts, and builds Markdown files you can use again.

Use it when you want to:
- capture colors, spacing, fonts, and layout rules
- build a clean style guide from an existing page
- create validation demos for AI-made pages
- keep your UI style in one place

It is made for people who want a fast way to study a design and reuse it.

## 🎨 What it extracts

ui-style-extractor focuses on common design parts that help you reuse a page style.

It can collect:
- color values
- font choices
- spacing rules
- border and radius styles
- button and form patterns
- layout structure
- section order
- simple visual notes

It then turns that data into Markdown files that are easy to read and share.

## 🧰 Troubleshooting

If you do not see a result file:
- check the output folder
- make sure the source page loaded
- try running the app again
- verify that you have write access to the folder

If the style guide looks off:
- use a cleaner source page
- remove popups or overlays before extraction
- try a page with fewer moving parts
- check that the source uses the styles you want to capture

If Windows blocks the app:
- open the release page again
- confirm you downloaded the latest file
- right-click the file and check Properties
- if the file is marked as blocked, unblock it and run again
